# Pomodoremi

## Installation

Make sure [RabbitMQ](http://www.rabbitmq.com/) server is up and
running.

```shell
npm install pomodoremi --global
```

## Usage

```shell
pomodoremi server

pomodoremi start
pomodoremi start Test
pomodoremi shortBreak
pomodoremi longBreak
pomodoremi stop
pomodoremi tag
```
